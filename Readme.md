# Mine Imator Simply Remade: Nuxi

For all your Minecraft animation needs!

## Dev environment

Requirements

- Dotnet 9.0+
- [This](https://drive.google.com/file/d/1iLjB1zmbnLmsFVDjbBHVVxI9lXo5fYxN/view?usp=sharing) Godot build

## Setup

1. Clone the repo
2. Download the Godot build
3. Run `dotnet nuget add source <folder/path/to/GodotNuget>` --name GodotNuget
4. Open the project in Godot
5. Run the project
